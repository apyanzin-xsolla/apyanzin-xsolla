## About Me

I am a Product Owner focused on platform products, developer ecosystems, experimentation, and analytics.

Right now my work is connected with Shop Builder: a platform direction for building and customizing game commerce experiences. The long-term idea is close to **Shopify Apps for the gaming industry**: a flexible foundation where internal teams, game developers, and third-party creators can build, extend, and reuse commerce solutions faster.

## What I Build

| Direction | Focus |
| --- | --- |
| Platform strategy | Shared foundations, extensibility, adoption, self-service growth |
| Developer experience | APIs, SDKs, docs, sandbox flows, onboarding paths |
| No-code and low-code | Customization systems for business users, designers, and integrators |
| Experimentation | A/B testing, metrics design, decision quality, analytics loops |
| Game commerce | Web shops, partner enablement, commerce tooling, launch speed |

## Product Operating System

| Step | Output |
| --- | --- |
| 1. Define the user problem | Clear segment, pain, expected behavior change |
| 2. Prototype fast | Small working version before heavy planning |
| 3. Measure behavior | Practical metrics, not vanity dashboards |
| 4. Iterate with evidence | Keep what improves the system, remove what does not |

## Metrics I Care About

| Metric | Formula |
| --- | --- |
| Time to first live site | `first_successful_publish_at - project_created_at` |
| Self-service adoption | `self_service_launches / total_launches` |
| Publish conversion | `published_sites / created_sites` |
| Integration time | `production_ready_at - integration_started_at` |
| Experiment velocity | `completed_experiments / month` |
